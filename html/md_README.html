Chromat-Ynk

Implementation of the Chromat-Ynk project by Efe Tatar, Nathan Signoud, Alize Fortunel, Francois Bretagne. The concept behind the project belongs to Cy Tech who charged the authors with its implementation.

This project is composed of two modules, the language interpreter and the interface.

Interpreter

The Abbas language interpreter takes in a .abbas file and interprets it. It implements variables, functions, arithmetic operations and boolean operations. Although the language is standalone, it has additionnal features when linked to an interface. Theses features include managing cursors enabling the user to draw lines on a canvas. Because of the nature of the interpreter, the user can manage to draw smooth shapes such circles, squares, sinusoidal curves and much more.

A simple guide to the abbas language

All files must end with the .abbas extention.

Declaring variables:
There are 3 types available: NUM, STR, BOOL
{type} {name} [ = {value} ] ;

Declaring loops:
There are 3 loops available: IF, WHILE, FOR
{type} condition { body }

Declaring functions:
DEF {name} { body }

Calling functions:
functions do not pocess strict parameters but take arguments instead. Let exp be a function, in order to call exp, one must:
exp(operation, operation, operation, ...);
There parameters will be available to the user as arg0, arg1, ...
The variable argc is automatically created and stores the number of arguments<br>

Return values
Function can return values at any given moment with the keyword RETURN
the implicit main function can infact return a value.

How to draw ?
The user has access to the following instructions:
FWD value [%]
BWD value [%]
TURN value
MOV x [%], y [%]
POS x [%], y [%]
HIDE
SHOW
PRESS value [%]
COLOR #RRGGBB
COLOR red, green,blue
THICK value
LOOKAT cursorID
LOOKAT x [%], y [%]
CURSOR cursorID
SELECT cursorID
Note that these instruction must be called as if they were functions.

Interface

The Interface enables the user to interpret .abbas files and visualise outputs. The user can choose to visualise the drawing process either step by step or in a constant speed.
