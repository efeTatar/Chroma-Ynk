\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classcom_1_1tsan_1_1chromaynk_1_1Context}{Context}} class acts like memory for the Abbas language interpreter~\newline


It contains variables, cursors, functions and the return value.~\newline
 It also has access to the canvas if the interpreter has to execute instructions
