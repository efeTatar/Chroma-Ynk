\doxysubsection{Detailed Description}
The token iterator is an essential tool for the parser~\newline


It can return the previous, current and next tokens in the list~\newline
 While returning the previous and following tokens, it increments or decrements the index~\newline
 It returns a EOF token if the index is out of range
