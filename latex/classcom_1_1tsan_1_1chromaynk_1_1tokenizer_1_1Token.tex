\doxysubsection{Detailed Description}
\mbox{\hyperlink{classcom_1_1tsan_1_1chromaynk_1_1tokenizer_1_1Token}{Token}} class~\newline


A token contains a value and a type~\newline
 The value is a string to be interpreted by the parser~\newline
 Types are defined in the token\+Type enumeration~\newline


The tokenizer does not handle negative values~\newline
 The parser handles theem during operation parsing.
